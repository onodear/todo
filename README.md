# TODOアプリ

非常にシンプルな TODO 管理アプリです。  
勉強のために作成したもので、フロント（HTML / JavaScript）と Node.js で作成した簡易的な API を組み合わせて動作します。

## 機能
- TODO の追加
- TODO の削除
- 期限の登録
- フロントから fetch でデータを送信
- API 側の DB でデータを保持

## 使用技術

### フロントエンド
- HTML
- JavaScript（DOM 操作、fetch API）

### バックエンド
- Node.js
- Express
- REST API（GET / POST / DELETE）
